# Simple Interest Calculator

This project is a simple interest calculator created using Bash scripting.

## Formula Used

Simple Interest = (P × T × R) / 100

Where:
- P = Principal amount
- T = Time
- R = Rate of interest

## Features
- Calculates simple interest
- Easy to use
- Command-line based

## Example

Input:
Principal = 1000
Rate = 5
Time = 2

Output:
Simple Interest = 100

## How to Run

Run the following command:

bash simple-interest.sh
